About the Project

This project replaces the traditional paper-based registration system with a fingerprint-based solution. By scanning a fingerprint, the student's details are instantly retrieved and logged into a web interface. This makes the process secure, fast, environmentally friendly, and eliminates errors associated with manual entry.

Technologies used include: fingerprint scanning hardware, HTML/CSS/JavaScript for the frontend, and optional backend for storage and management.
